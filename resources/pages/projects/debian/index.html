Debian repository

For some selected packages (syslog-ng 3.3, Zorp GPL and dependencies as of this writing), I provide up-to-date packages for a number of Debian and Ubuntu releases (see the select box just below). The packages will be kept up to date for all platforms, until at least after a month of the platform's end of life. In case of syslog-ng, to see what changed between versions, please see its changelog.

In addition to the above, to fully utilise the power of Zorp, one needs a patched kernel too, which are available (at least 2.6 kernels, for older distributions, with 3.x kernels in the works) in a separate repository: 

deb      http://packages.madhouse-project.org/zorp-kernel kernel 2.6

The archive is signed by GPG key 0x6CE70C6E8B9E0644, which one can add to apt as this: 

curl http://packages.madhouse-project.org/debian/archive-key.txt | sudo apt-key add -

The packages are available for the i386 and amd64 architectures on both Debian and Ubuntu, but in addition to that two, the Debian repositories also contain syslog-ng packages for the powerpc and kfreebsd-amd64 architectures. If there's an interest, I can pull in more build machines and compile for other platforms aswell.
